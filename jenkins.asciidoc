Installation
------------

  cat > /etc/apt/sources.list.d/jenkins.list << EOF
  deb http://pkg.jenkins-ci.org/debian binary/
  EOF

  wget -q -O - http://pkg.jenkins-ci.org/debian/jenkins-ci.org.key | apt-key add -
  apt-get update
  apt-get install jenkins
